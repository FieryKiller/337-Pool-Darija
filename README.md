# 1337 Quest: The C Piscine Pool 🦾🏆

A premium, gamified 1337/42 Network training simulator designed in **Darija/English** to help students master the C Programming Pool effortlessly.

![Screenshot Simulator](https://raw.githubusercontent.com/FieryKiller/337-Pool-Darija/main/preview.png)

## 🌟 Key Features
- **21-Level Roadmap**: Sequential quests covering Day 00 Shell to C10 File Descriptors.
- **Integrated Lesson Guide**: Full documentation (Summaries, Concepts, Q&A) inside the IDE.
- **Terminal Simulator**: Real-time compilation simulation with Norminette fakes.
- **Progression System**: Gain Score, Unlock Ranks (from Pisciner to Legend), and Save Stats.
- **Pathfinder Onboarding**: Choose your pool path (Shell, C, or Full).
- **PRO Aesthetic**: High-end glassmorphism, dark theme, and magic animations.

## 🛠️ Tech Stack
- **Languages**: HTML5, CSS3 (Vanilla), JavaScript (ES6).
- **Icons**: FontAwesome 6.
- **Fonts**: Google Fonts (Outfit, Consolas).
- **Persistence**: LocalStorage (Zero-server required).

## 📁 Project Structure
- `index.html`: Dashboard & Simulator UI.
- `style.css`: Design System & Animations.
- `app.js`: Game Logic & Terminal Engine.
- `curriculum_data.js`: Full 24-chapter Pool Guide.

## 🚀 How to Run Locally
1. Clone the repo: `git clone https://github.com/FieryKiller/337-Pool-Darija.git`
2. Open `index.html` in your browser.
3. Start your 1337 journey.
